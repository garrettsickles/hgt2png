# hgt2png

![Build Status](https://travis-ci.org/garrettsickles/hgt2png.svg?branch=master)

Convert HGT raster to a subset PNG rasters

## Usage

```
./hgt2png <HGT FILE>.hgt 3601 3601 <PNG FILE>.png
```

## Example
```
./hgt2png N36W113.hgt 3601 3601 N36W113.png
```
